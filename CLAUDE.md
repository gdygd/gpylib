# Python Project Development Guide

This project uses **Poetry** for dependency management and virtual environment handling.

## Python Version

Python version is managed via pyproject.toml.

Required:
>=3.10,<4.0

## Environment Setup

Install dependencies:

    poetry install

This will automatically create a virtual environment and install all dependencies.

## Running Python Code

Use poetry to run all Python commands.

Example:

    poetry run python examples/example.py

Do NOT run python directly outside the poetry environment.

## Adding Dependencies

Runtime dependency:

    poetry add <package>

Development dependency:

    poetry add --group dev <package>

Example:

    poetry add requests
    poetry add --group dev pytest

## Running Tests

    poetry run pytest

## Code Lint / Format (if configured)

    poetry run ruff check .
    poetry run ruff format .

## Virtual Environment

Check environment info:

    poetry env info
